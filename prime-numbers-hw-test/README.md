# Prime numbers hardware test

## Prerequisites

Download the [original project](https://github.com/kritikov/Fully-Parallel-Prime-Number-Search-on-FPGA.git).

Copy the VHD files in the directory of this project.

## Build & load

```bash
make clean
make
make load
```
